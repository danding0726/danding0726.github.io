当所有人以为 Nvidia 只会卖芯片时，它悄悄下场做软件了。

## NemoClaw 是什么

据 Wired 报道，Nvidia 将在下周 GTC 大会上正式发布开源 AI Agent 平台 NemoClaw。这个平台将允许企业软件公司派遣 AI Agent 完成多步骤任务——不管这些公司的产品是否运行在 Nvidia 的芯片上。

这是 Nvidia 战略的重大转变。过去十年，Nvidia 的商业模式建立在"卖硬件"的基础上：数据中心 GPU、游戏显卡、CUDA 生态系统。但 AI Agent 的崛起威胁到了这一王国的根基——如果 AI 能自己完成工作，企业还需要买那么多 GPU 吗？

Nvidia 的答案是：既然无法阻挡 Agent 浪潮，那就成为 Agent 浪潮的基础设施提供商。

## 为什么是企业软件

Nvidia 的目标客户不是普通消费者，而是企业软件巨头。

报道显示，Nvidia 已经接触了以下公司寻求合作：

- **Salesforce**：全球最大的 CRM 软件提供商
- **Cisco**：企业网络和协作巨头
- **Google**：搜索和云服务霸主
- **Adobe**：创意软件领导者
- **CrowdStrike**：网络安全新贵

这些公司有一个共同特点：它们都拥有大量企业客户，都面临将 AI 能力嵌入产品的压力，也都在寻找可以信赖的 Agent 基础设施。

对 Nvidia 来说，这意味着它可以从两个层面获利：
1. **软件层**：NemoClaw 平台本身，以及相关的安全和隐私工具
2. **硬件层**：这些企业客户部署 Agent 时需要的 GPU

## 为什么是开源

Nvidia 选择开源 NemoClaw，是一个深思熟虑的决策。

**降低采用门槛**：企业软件市场高度分散，没有哪家供应商能通吃。如果 NemoClaw 收费或闭源，竞争对手可以轻松绕过。开源意味着"没有护城河，但也没有壁垒"——任何人可以免费使用，但 Nvidia 仍然可以通过企业级支持和服务盈利。

**对抗 OpenAI**：OpenClaw 已经被 OpenAI 收购，成为其进军 Agent 市场的武器。如果 Nvidia 不做出回应，开源 Agent 领域的话语权将完全掌握在 OpenAI 手中。

**建立标准**：谁主导了开源 Agent 平台，谁就可能成为事实标准的制定者。这对 Nvidia 的长期芯片业务有战略价值。

## 企业市场的冰与火

AI Agent 在企业环境中正处于一个微妙的位置：

**热情这一面**：企业迫切需要自动化。客服工单处理、CRM 数据录入、财务对账、HR 招聘筛选——这些都是 Agent 的天然应用场景。Gartner 预测，到 2028 年，40% 的企业应用程序将内置 AI Agent。

**恐惧这一面**：安全问题让人夜不能寐。上个月，一位 Meta 员工公开讲述了一个 AI Agent 在她电脑上"失控"、批量删除邮件的故事。多家科技公司已经要求员工不要在工作电脑上使用开源 Agent 工具。

Nvidia 的策略是：用"安全"和"隐私"作为卖点。报道提到，Nvidia 计划在 NemoClaw 中提供企业级安全工具，解决 CIO 们的后顾之忧。

## 这是一场什么样的战争

AI Agent 平台正在变成一个新的战场：

- **OpenAI**：通过收购 OpenClaw 获得先发优势
- **Anthropic**：在 Claude 中构建 Agent 能力
- **Nvidia**：以开源平台身份入局
- **Meta**：虽然禁止员工使用 OpenClaw，但也在开发自己的 Agent 技术
- **Google**：Gemini 生态正在整合 Agent 功能

这不仅仅是技术竞争，更是生态系统的竞争。谁能吸引更多开发者、集成更多企业应用、建立更强大的插件生态，谁就可能成为下一个"操作系统"。

## 写在最后

Nvidia 卖芯片卖了三十年，现在它想证明自己也能做软件。

10 年前，没有人能想到一家显卡公司会成为 AI 时代的中心。10 年后，如果 NemoClaw 成功了，我们可能需要重新定义 Nvidia 是什么——不再只是一家芯片公司，而是一个企业 AI 基础设施平台。

这场反击战，才刚刚开始。
